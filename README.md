📌 Overview

This project focuses on analyzing e-commerce sales data to extract meaningful business insights and improve decision-making. The goal is to help an online business understand customer behavior, product performance, and revenue trends through data analysis and visualization.

The project covers the complete data analytics pipeline — from data cleaning to dashboard creation — enabling stakeholders to gain a clear understanding of sales patterns and profitability.

🎯 Objectives
Analyze overall sales and profit performance
Identify top-performing states and products
Understand customer purchasing behavior
Analyze preferred payment methods
Track monthly sales trends
Build an interactive dashboard

🛠️ Tech Stack
Python → Data Cleaning & Preprocessing
Microsoft Excel → Business Insights & Analysis
Power BI → Dashboard & Visualization

📂 Project Workflow
1. Data Cleaning (Excel)
Handled missing and inconsistent data
Removed duplicates
Standardized formats for analysis

3. Data Analysis and business insights (Excel)
Created pivot tables
Performed aggregations and filtering
Derived key business insights

5. Data Visualization (Power BI)
Built an interactive dashboard with:
Sales by State
Sales by Product Category
Profit by Category
Payment Mode Analysis
Monthly Sales Trends
Top Products by Revenue
